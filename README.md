## Real-Time Ride Booking Data Pipeline

Real-Time Ride Booking Data Pipeline is an end-to-end real-time data engineering pipeline for a ride booking platform. Streaming ride events are ingested through Azure Event Hubs, processed using Databricks Structured Streaming with PySpark, and stored in a Delta Lake Medallion architecture (Bronze, Silver, Gold). The curated datasets are modeled into a star schema for analytics-ready reporting.
