## RideBookingEvent_DataEngineer_Project
RideBookingEvent_Data_Engineer_Project is an end-to-end cloud data engineering project simulating a real-time ride booking platform. Ride events stream through Kafka, are processed with Databricks (Apache Spark), and stored in Azure Data Lake for analytics and reporting, demonstrating scalable event pipelines and modern data engineering practices.
